# Shop-Shop

A mobile shopping app built using React Native and Expo.

## 📱 Features

- Browse products
- View product details
- Add to cart

## 🛠️ Tech Stack

- React Native
- Expo

## 🚀 Getting Started

```bash
git clone https://github.com/wondmeneh123/Shop-Shop.git
cd Shop-Shop
npm install
npx expo start
